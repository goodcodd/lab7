# Аналіз результатів та звітність

Після реалізації програмного модуля `PasswordValidator` та створення набору модульних тестів було виконано запуск тестового набору за допомогою фреймворку **Jest**. Тестування проводилося для перевірки коректності роботи методів класу при допустимих, некоректних та граничних вхідних даних, що відповідає вимогам лабораторної роботи.

Для запуску тестів використовувалась команда:

```bash
npm test
```

Для отримання звіту про покриття коду використовувалась команда:

```bash
npm run test:coverage
```

У результаті виконання тестового набору було отримано такі показники:

- **Test Suites:** 1 passed, 1 total
- **Tests:** 27 passed, 27 total
- **Snapshots:** 0 total
- **Time:** 0.887 s

Отже, усі розроблені тести були успішно виконані, помилок типу **Failures** під час запуску не виявлено. Це свідчить про те, що реалізований модуль коректно обробляє основні сценарії використання, у тому числі позитивні, негативні та граничні випадки.

## Аналіз звіту про покриття коду

Після запуску тестів із параметром `--coverage` було сформовано звіт про покриття коду. Отримані результати:

- **Statements:** 97.91%
- **Branches:** 97.72%
- **Functions:** 100%
- **Lines:** 97.5%

Отримані значення свідчать про високий рівень покриття програмного модуля тестами. Усі функції класу були виконані під час тестування, а більшість логічних гілок були перевірені окремими тестовими сценаріями. Неповне покриття зумовлене тим, що одна з гілок у коді не була протестована окремим тестом, хоча основна функціональність модуля перевірена повністю.

## Висновок за результатами тестування

У ході виконання лабораторної роботи було налаштовано середовище тестування для JavaScript із використанням фреймворку **Jest**, реалізовано програмний модуль для перевірки складності паролів та створено повний набір модульних тестів. Проведений аналіз показав, що модуль працює коректно, а тестовий набір забезпечує високе покриття коду та перевіряє систему на коректних, помилкових і граничних вхідних даних. Таким чином, поставлене завдання лабораторної роботи виконано успішно.
